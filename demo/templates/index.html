Data Visualization

This is a proof of concept of visualizing data using the Django framework and Seaborn library. Seaborn is a powerful visualization library that is built on top Matplotlib library. It is easy to use and does not require many lines of code.

Matplotlib supports formats like SVG and PNG, that are an excellent way to send reports and can be easily be embedded into emails.

The Data Set

For this demo, we will use a synthetic data set which contains models: Transaction and Store.
The Store model, represents a physical store. It has a location (state), number of employees, store size and expenses (in USD).
The Transaction model, contains information about one transaction that took place in a store. It holds information about the transaction date, the transaction amount and the store where the transaction took place.
It is important to mention that all the data was generated using a random number generator and thus the charts might not reflect the reality.
